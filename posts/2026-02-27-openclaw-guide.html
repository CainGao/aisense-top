📖 OpenClaw从零开始：5分钟搭建你的AI私人助理

2026年02月27日 作者：功哥

🎯 目标

5分钟内完成OpenClaw的安装和基本配置，打造你的第一个AI私人助理。

📋 准备工作

  • macOS / Linux / Windows 系统
  • Node.js 18+ 已安装
  • 基本的命令行操作能力

🚀 安装步骤

步骤1：安装OpenClaw

npm install -g openclaw

步骤2：初始化配置

openclaw init

按照提示完成基本配置。

步骤3：启动Gateway

openclaw gateway start

Gateway是OpenClaw的核心服务，负责AI模型调用和消息路由。

步骤4：配置AI模型

openclaw configure --section ai

输入你的AI API密钥（OpenAI、Anthropic等）。

步骤5：测试运行

openclaw status

如果看到"Gateway: running"，说明安装成功！

💡 第一个任务

尝试让AI帮你做一件事：

openclaw ask "帮我写一个Python脚本，计算斐波那契数列"

🔧 常用命令

  • openclaw status - 查看系统状态
  • openclaw gateway start - 启动Gateway
  • openclaw gateway stop - 停止Gateway
  • openclaw configure - 配置系统
  • openclaw cron add - 添加定时任务
