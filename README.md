## 简介

AI Sense 是一个中文AI工具导航站，每日精选热门AI工具，提供分类导航和深度评测。

## 部署

本站为纯静态网站（HTML + CSS），部署在 GitHub Pages。

- 地址: [aisense.top](https://aisense.top)
- 备用: [caingao.github.io/aisense-top](https://caingao.github.io/aisense-top)

## 技术栈

- 纯 HTML5 + CSS3
- 无框架依赖
- 响应式设计，移动端适配
- 暗色主题
