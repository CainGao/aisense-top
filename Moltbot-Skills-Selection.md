## 🛠️ 安装指南

### 前提条件

- Moltbot 系统已配置
- Python 3.x 已安装
- Git 已配置
- 相关 API 密钥已获取（如需要）

### 安装步骤

#### 步骤 1：访问 GitHub 仓库

```bash
访问：https://github.com/VoltAgent/awesome-moltbot-skills
```

#### 步骤 2：浏览技能列表

- 查看技能分类（资讯、开发工具、创意、社交媒体）
- 查看技能描述和文档
- 查看技能评分和推荐指数

#### 步骤 3：选择技能

- 根据优先级选择技能
- 记录技能名称和 ID
- 记录技能安装依赖

#### 步骤 4：安装技能

```bash
# 示例：安装 web-search 技能
moltbot skills install web-search

# 示例：安装 git-integration 技能
moltbot skills install git-integration

# 示例：安装 yuque-integration 技能
moltbot skills install yuque-integration
```

#### 步骤 5：配置技能

```bash
# 示例：配置 web-search 技能
moltbot skills configure web-search

# 示例：配置 git-integration 技能
moltbot skills configure git-integration

# 示例：配置 yuque-integration 技能
moltbot skills configure yuque-integration
```

#### 步骤 6：测试技能

```bash
# 示例：测试 web-search 技能
moltbot skills test web-search

# 示例：测试 git-integration 技能
moltbot skills test git-integration

# 示例：测试 yuque-integration 技能
moltbot skills test yuque-integration
```

## ⚠️ 注意事项

### 1. 安全性
- 不要将 API 密钥提交到 Git 仓库
- 使用环境变量或配置文件存储敏感信息
- 定期轮换 API 密钥

### 2. 兼容性
- 确认技能与当前 Moltbot 版本兼容
- 确认技能依赖已安装
- 确认技能符合 Anthropic Agent Skill 开放标准

### 3. 性能
- 技能可能影响 Moltbot 响应速度
- 建议在生产环境测试性能
- 建议配置超时和重试机制

### 4. 成本
- 某些技能可能产生 API 调用费用
- 建议监控 API 调用量和费用
- 建议设置费用预警
