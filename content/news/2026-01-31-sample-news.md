# AISENSE - AI 资讯

## 大模型动态

### GPT-5 预览版发布
OpenAI 发布了 GPT-5 预览版，支持更长上下文和更快推理速度。预览版主要面向开发者和企业用户，API 定价尚未公布。

### Claude Code 2.2 更新
Anthropic 发布了 Claude Code 2.2，新增了 MCP 协议支持和代码审查增强功能。开发者现在可以直接在编辑器中调用外部工具。

### Google Gemini 2.0 Flash
Google 发布了 Gemini 2.0 Flash，专注于更低延迟和更快推理。模型在移动设备上的性能有显著提升。

---

## Agent 开发

### LangChain v0.1.3 发布
LangChain 发布了 v0.1.3 版本，新增了多 Agent 协作功能和 Memory 优化。开发者可以更容易构建具有多个 Agent 的复杂系统。

### AutoGPT v0.2.0 版本
AutoGPT 发布了 v0.2.0，添加了 GPT-4o 集成和自主任务执行能力。新版本支持更复杂的 Agent 工作流。

### CrewAI 1.0 发布
CrewAI 正式发布 1.0 版本，提供企业级 Agent 平台。新版本支持多 Agent 编排和分布式执行。

---

## AI 商业化

### OpenAI 推出企业版
OpenAI 宣布推出企业版，提供更高的 API 调用限制和优先支持。企业版面向大公司，定价为 $60/用户/月。

### Anthropic 发布 Claude API v2
Anthropic 发布了新的 API v2，提供更一致的模型行为和更好的错误处理。新 API 支持流式响应和批量请求。

---

## 工具与框架

### Vercel AI SDK
Vercel 发布了 AI SDK，简化了 AI 应用部署流程。开发者可以使用 SDK 快速部署 AI 应用到 Vercel 平台。

### Supabase AI Functions
Supabase 发布了 AI Functions 功能，支持直接在边缘计算环境运行 AI 推理。
