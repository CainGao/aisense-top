## LangChain

**工具类型**：Agent框架/LLM应用框架

**简介**：
LangChain 是一个用于构建 LLM 应用的框架，提供组件化和可扩展的开发体验。它支持 Chain、Agent、Memory、Tool 等核心概念。

**核心功能**：
- Chain 构建（顺序链、并行链、循环链）
- Agent 协作和编排
- Memory 管理（短时、长时、向量数据库）
- Tool 定义和调用
- 多种 LLM 提供商支持

**适用场景**：
- 构建具有复杂推理能力的 Agent
- 链接多个 LLM 调用
- 实现对话式 AI 应用
- 构建文档问答系统

**技术栈**：
- Python
- JavaScript/TypeScript
- 支持多种数据库（PostgreSQL、Redis、Pinecone 等）
